# Godot Radionica - Breakout Igrica

## Setup

### Instalacija Git-a

**Windows:**
1. Skinuti Git sa https://git-scm.com/download/win
2. Pokrenuti installer i pratiti uputstva (default opcije su ok)

**Linux:**
```bash
sudo apt install git
```

**Mac:**
```bash
brew install git
```

### Kloniranje repozitorijuma

```bash
git clone https://github.com/mark00vka/godot-radionica.git
cd godot-radionica
```

### Kako pratiti korake radionice

Svaki korak radionice ima svoj branch. Da se prebacite na odredjeni korak:

```bash
git checkout ime-brancha
```

Da vidite sve dostupne brancheve:

```bash
git branch -a
```
